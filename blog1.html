My First Experience with Linux: A Step-by-Step Guide to install Fedora

--by Rohith Sharma

Published on 20th August 2024 in blog post

As a Windows and macOS user, I was skeptical about trying out Linux at first. But, after hearing rave reviews from friends and colleagues who swear by its flexibility and customization options, I decided to take the plunge. In this blog post, I’ll share my first-hand experience with Linux, specifically Fedora, and provide a step-by-step guide on how to get started.

What is Linux?

Linux is an open-source operating system that has gained immense popularity among developers, hackers, and tech enthusiasts. It’s free, customizable, and secure, making it an attractive alternative to Windows and macOS. In simple terms, Linux is the software that manages your computer’s hardware and resources, allowing you to run applications and interact with files.

Started:

To try out Linux, I chose Fedora as my distro of choice. I downloaded the ISO file from the official website and created a bootable USB stick using Rufus on Windows. This process is relatively straightforward and can be done using various tools like Ubuntu or the built-in Disk Utility on MacOS.

The Boot Process:

To boot Linux from your USB drive, make sure to disable Secure Boot in your BIOS menu (F10 for me, on my HP Probook). Once you’ve done that, insert the USB stick and restart your computer. You’ll be presented with a boot menu where you can select the Linux option.

Installing Fedora:

After successfully booting into Linux, I decided to install Fedora alongside Windows. The installation process is relatively simple and self-explanatory. You can choose to customize various settings, such as the desktop environment, package selection, and user accounts.

Customizing Your Experience:

One of the best things about Linux is its customization options. As a fan of minimalism, I chose to install Fedora with a sleek, modern desktop environment that’s perfect for everyday use. You can also choose from a range of themes, fonts, and icons to personalize your experience.

What Next?

If you’re new to Linux like me, here are some next steps:

1. Explore the Command Line: Linux is all about command-line interfaces (CLI) and terminal commands. Take some time to learn the basics of using the CLI and mastering basic commands. 2. Discover New Tools: Linux has an incredible array of tools and software available for download. Explore the Software Center or online repositories like apt-get or pip. 3. Join Online Communities: Connect with other Linux enthusiasts on forums, Reddit, and social media to learn more about best practices, troubleshooting, and new releases. And it's fun too.

Conclusion

My first experience with Linux has been overwhelmingly positive. With Fedora, I’ve discovered a world of customization options, flexibility, and security that’s hard to find in Windows or MacOS. Whether you’re a seasoned pro or just starting, Linux is an excellent choice for anyone looking to expand their tech horizons.
